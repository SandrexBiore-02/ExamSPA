# Angular SPA Project

## Description
This project is a SPA demonstrating routing, forms, and TypeScript usage.

## Technologies Used
- Angular 17
- TypeScript
- Node.js
- GitHub

## How to Run

1. Clone repository
git clone https://github.com/yourusername/angular-spa-project.git

2. Install dependencies
npm install

3. Run project
ng serve

4. Open browser
http://localhost:4200
